def ask_wiki_for str
        list = false
        Thread.new do
                list = WikiClient.ask(str)
                draw_options list
        end
        # testing helper, remove this if the gui is used
        until list
                sleep 0.1
        end
end

    draw_options(params) click to toggle source

    Update the View of the options_list with the given informations. This method takes different arguments depending on the current use-case. When you just want to show your results,

    • ApplicationControllerInstance.draw_options list: […]

    But if you want to show a special page, use

    • ApplicationControllerInstance.draw_options site: “my_site”

    where “my_site” can be one of the following special sites:

    • "500" # => an Error Page

    • "404" # => Nothing found
