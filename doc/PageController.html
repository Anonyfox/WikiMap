new(params) click to toggle source

create a new page. expects a phrase and a link list. opens an existing page if there is one. The link list is optional. sample call:

  • pc = PageController.new phrase: my_phrase, links: [“link_1“, “link_2“]

in the :target_pages attribute stands an array of Page-Objects. if you just want the names, just use the :target_page_names attribute.
