    new(params) click to toggle source

    create a new page. expects a phrase and a link list. opens an existing page if there is one. The link list is optional. sample call:

    • pc = ::new phrase: my_phrase, links: [“link_1”, “link_2”]

    in the :#target_pages attribute stands an array of Page-Objects. if you just want the names, just use the :#target_page_names attribute.

    # File old_staff/app/controllers/page_controller.rb, line 15
def initialize params
        phrase = params[:phrase] || nil
        links = params[:links] || []
        return false unless phrase

        if page_exists? phrase
                @page = Page.where(phrase: phrase)[0]
                if @page.crawled
                        # TODO: recrawl here if old informations
                        @targets = @page.targets.split(",")
                else
                        @targets = add_target_pages(links)
                        @page.targets = @targets.join ","
                        @page.save
                end

        else #create a new page
                @page = Page.new phrase: phrase, crawled: true
                @targets = add_target_pages(links)
                @page.targets = @targets.join ","
                @page.save
        end
        @target_pages = lookup_target_pages
        @target_page_names = @target_pages.map{|pg| pg.phrase}
end

end
    add_uncrawled_page(name) click to toggle source

    add a new page with the givem phrase and return its id. If a page with this phrase is existing yet, just take this one and return its id.

    # File old_staff/app/controllers/page_controller.rb, line 51
def add_uncrawled_page name
        crawled_page = Page.where(phrase: name)[0]
        if crawled_page #take this one
                return crawled_page.id
        else #create a new page
                new_page = Page.new phrase: name, crawled: false
                new_page.save
                return new_page.id
        end
end
